# Personalized Caffeine Recommendation System

## Project Overview

This project investigates the cognitive and health effects of caffeine consumption based on individual differences such as age, sleep patterns, stress level, and lifestyle factors.

Using multiple datasets and machine learning techniques, the study analyzes how caffeine intake influences performance and health indicators, and proposes a personalized recommendation system for optimal daily caffeine consumption.

---

## Objectives

- Analyze the relationship between caffeine intake and cognitive performance
- Examine health-related indicators such as sleep quality and fatigue
- Apply statistical hypothesis testing
- Train regression and machine learning models
- Develop a personalized caffeine intake recommendation system

---

## Methods

- Data preprocessing and cleaning
- Feature engineering
- Exploratory Data Analysis (EDA)
- Statistical testing
- Machine Learning models (Regression-based approaches)
- Model evaluation and comparison

---

## Output

The system generates:

- Statistical analysis results
- Model performance metrics
- Personalized caffeine intake suggestions
